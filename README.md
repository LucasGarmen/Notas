# Mobile NoteApp 📱

Um aplicativo de bloco de notas leve, intuitivo e focado em produtividade pessoal, desenvolvido especificamente para dispositivos móveis.

## 📥 Como Baixar e Instalar (.APK)

Para testar o aplicativo direto no seu celular Android, siga os passos:

1. **Baixe o instalador:** Clique no link abaixo para baixar o arquivo mais recente:
   👉 [**Clique aqui para baixar o Mobile-NoteApp.apk**](LINK_DE_LA_RELEASES_AQUI)
2. **Abra o arquivo:** Após o download, clique no arquivo `.apk` para iniciar a instalação.

### ⚠️ Nota Importante sobre Segurança (Play Protect)
Como este aplicativo é um projeto independente de código aberto e **não está publicado na Google Play Store**, o sistema Android (Google Play Protect) exibirá um alerta informando que o app é de uma "fonte desconhecida".

> 🔒 **O aplicativo é 100% seguro.** Todo o código-fonte está disponível publicamente neste repositório para consulta e auditoria. Para prosseguir, basta clicar em **"Instalar assim mesmo"** (ou permitir a instalação de fontes desconhecidas nas configurações do aparelho).

## 🛠️ Descrição do Produto & Funcionalidades

O **Mobile NoteApp** foi estruturado pensando na agilidade do dia a dia, garantindo uma interface limpa e respostas rápidas.

* **Interface Responsiva:** Design totalmente adaptado para telas de smartphones, priorizando a ergonomia e o uso com apenas uma mão.
* **Gerenciamento CRUD Completo:** Permite criar, visualizar, editar e excluir notas de forma rápida e dinâmica.
* **Persistência de Dados Local:** As notas são salvas no armazenamento interno. Isso garante acesso total às informações e criação de notas mesmo **offline** (sem internet).
* **Desempenho Otimizado:** Consumo mínimo de memória e processamento, rodando de forma fluida em qualquer hardware Android.
